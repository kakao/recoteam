- Summary : 3줄 내외로 논문 내용을 요약합니다.
- Approach : 논문에서 쓰인 접근법들을 정리합니다.
- Results : 실험 결과에 대한 내용을 정리합니다.
- Conclusion : 논문에 대한 본인의 생각을 정리합니다.
